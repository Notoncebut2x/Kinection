PROJECT OVERVIEW
----------------
A web platform for personal DNA analysis against the Allen Ancient DNA Resource
(AADR) v62.0 — 17,629 ancient individuals across ~406,000 SNPs. Users upload
raw AncestryDNA / 23andMe files and receive haplogroup assignments, ancient
population matches, PCA positioning, and a narrative ancestry report.

Methodology mirrors the Deep Maniot Greeks study (Communications Biology, 2026).

Reference dataset:  AADR v62.0 (17,629 individuals, 1,233,013 SNPs in GENO file)
Modern individual:  AncestryDNA V2.0 (660,192 SNPs)
Dataset breakdown:  9,601 male / 7,269 female / 15,707 PASS assessment
                    15,892 with Y haplogroup / 12,477 with mtDNA haplogroup

STEP 1.1 — Data Parsing and Harmonisation  [COMPLETE, run 2026-04-22]
File: scripts/step1_parse_harmonise.py

Why we're doing it:
  Your DNA test gives you ~660,000 genetic markers, while the ancient dataset
  uses a different set of ~400,000 markers. Before any comparison is possible,
  we need to find the markers that appear in both lists, make sure they're
  being read from the same strand of the DNA double helix, and convert
  everything into a common numerical format. Think of it like translating two
  documents written in slightly different dialects into a single shared language
  before you can compare them. This step produces the shared "dictionary" that
  every subsequent analysis is built on — nothing else can run without it.

What it does:
  - Parses AncestryDNA raw file: 660,192 SNPs across all chromosomes
      (chr 1-22, X, Y: 1,628 SNPs, MT: 190 SNPs, PAR: 519 SNPs)
  - Opens the EIGENSTRAT binary .geno file (2 bits per genotype, mmap'd)
  - Parses the .ind individual list and .anno annotation file
  - Finds the SNP position overlap between modern and ancient datasets
  - Performs strand alignment — excludes palindromic (A/T, C/G) SNPs
  - Encodes modern genotypes as alt-allele dosage (0/1/2, -1=missing)
  - Writes encoded genotypes as a numpy .npy array

Key result: 394,671 overlapping SNPs found between modern individual
            and the ancient dataset (after strand alignment filtering)

Outputs:
  output/step1_rn/snp_overlap.tsv         — SNP positions, rsID, dosage
  output/step1_rn/modern_indv_rn_encoded.npy — dosage array (int8)
  output/step1_rn/step1_summary.json       — run statistics
  output/step1_rn/step1.log

Tools/libraries used:
  - Python 3.14
  - NumPy (array encoding)
  - Custom EIGENSTRAT binary parser (GenoFile class in utils/parsers.py)
  - Custom AncestryDNA parser (parse_ancestry_dna in utils/parsers.py)
  - Custom ANNO parser (parse_anno_file in utils/parsers.py)


STEP 1.2 — Haplogroup Assignment  [COMPLETE]
File: scripts/step2_haplogroup.py

Why we're doing it:
  Your paternal lineage (passed down through men, father to son) and your
  maternal lineage (passed down through women, mother to child) are tracked by
  two special parts of your DNA — the Y chromosome and mitochondrial DNA.
  These don't get reshuffled every generation like the rest of your genome,
  so they act like a family surname for your deep ancestry. By identifying
  which branch of the global family tree you belong to and then finding ancient
  individuals who share that same branch, we can pinpoint exactly which ancient
  populations your direct paternal and maternal lines descend from, and roughly
  when those lineages lived. This is the most direct, unambiguous ancestry
  signal we can extract.

What it does:
  - Assigns Y-DNA (paternal) haplogroup using ISOGG marker database
      Checks derived state at each Y marker; walks tree from root to tip
      Resolves rsID vs genomic position for matching
  - Assigns mtDNA (maternal) haplogroup using PhyloTree B17 logic
      Scores candidate haplogroups by proportion of required markers found
  - Searches all 17,629 ancient individuals for haplogroup matches
      Ranks by match type (Y+MT > Y-only > MT-only) and proximity score
  - Computes allele-sharing distance (ASD) on Y chromosome vs all ancient males
  - Computes ASD on mitochondrial DNA vs all ancient individuals

STEP 1.3 — Genome-wide SNP Similarity and PCA  [COMPLETE]
File: scripts/step3_similarity_pca.py

Why we're doing it:
  Haplogroups only tell you about two thin lines of ancestry — one through
  your father's father's father... and one through your mother's mother's
  mother. But you also inherited DNA from hundreds of other ancestors. This
  step looks at your entire genome and asks: across all 394,000 shared genetic
  markers, which ancient people look most similar to you overall? It's the
  difference between knowing your surname came from Ireland versus knowing
  that 40% of your ancestry traces to Bronze Age Iberia. The PCA part creates
  a kind of genetic map — imagine plotting every ancient person as a dot on a
  page based on their DNA, and then dropping your own dot in to see which
  clusters you land nearest to. That's the most powerful tool in modern
  population genetics for visualising where someone sits in the ancient world.

What it does:
  - Computes pairwise allele-sharing distance (ASD) between the modern
    individual and all 17,629 ancient individuals
  - Uses pseudo-haploidisation: at heterozygous sites, randomly picks one
    allele (10 draws averaged) so both sides are pseudo-haploid — eliminates
    the systematic 0.5 bias that inflates distances with naive diploid/haploid
    comparisons
  - Autosomal chromosomes only (chr 1-22); processes in 5,000-SNP chunks
  - Aggregates individual distances to population-level means
  - Identifies high-coverage ancient individuals for PCA (≥100,000 1240k SNPs)
  - Computes PCA via scikit-learn; projects modern individual into ancient space
  - Writes population rankings and individual rankings

PCA variance explained (10 components):
  PC1: 3.04%  PC2: 2.90%  PC3: 0.79%  PC4: 0.42%  PC5: 0.30%
  PC6: 0.27%  PC7: 0.17%  PC8: 0.15%  PC9: 0.14%  PC10: 0.13%

Outputs:
  output/step3_rn/pairwise_distances.tsv     — 17,629 individual distances
  output/step3_rn/population_distances.tsv   — 3,412 populations ranked
  output/step3_rn/pca_coordinates.tsv        — ancient + modern PCA coords
  output/step3_rn/pca_variance_explained.json
  output/step3_rn/top_matches_report.md       — narrative report
  output/step3_rn/step3.log

Tools/libraries used:
  - Python 3.14
  - NumPy (vectorised ASD, chunk processing)
  - scikit-learn PCA (with numpy SVD fallback if sklearn unavailable)
